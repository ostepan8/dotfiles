⚡Quick start

The 8 shortcuts you'll use every day. Memorize these first.

Opt+Space
New Ghostty terminal window
Opt+`
Drop-down quick terminal
Alt+1 … 9, 0
Jump to workspace 1–10
Alt+Shift+N
Send current window to workspace N
Alt+[ / ]
Focus laptop / external monitor
Alt+\
Move window to other monitor (+ follow)

Atuin — fuzzy search shell history
Cmd+Shift+A
Chrome: fuzzy search every tab

🚀Launch apps — skhd

Global hotkeys. Work from anywhere, any app. All use Opt as leader.

Opt+Space
New Ghostty window — lands on your current workspace
Opt+`
Drop-down quick terminal — temporary, slides from top. Press again to hide.
Opt+B
Chrome — personal profile (ohstep23@gmail.com)

Chrome — work profile (owen@subconscious.dev)
Opt+S
Slack
Opt+M
Spotify
Opt+E
New Finder window

🗂️Workspaces — Aerospace

10 virtual desktops. Each holds its own tiled windows. Active workspace shows in orange on the top bar.

Alt+1 … 9
Jump to workspace 1–9
Alt+0
Jump to workspace 10
Alt+Tab
Go to previous workspace (last-used)
Alt+Shift+1 … 9, 0
Send current window to workspace N (you stay put)

Suggested layout

1
Code (Cursor + Ghostty)
2
Browser
3
Chat (Slack + Messages)
4
Music / misc
0
Scratch

Arrange externals to the left of main in System Settings → Displays. Prevents flickering.

Alt+[
Focus laptop (built-in display)

Focus external monitor
Alt+\
Move current window to other monitor + follow it there

🪟Windows — Aerospace tiling

When multiple apps are on one workspace, they tile automatically. Navigate with vim keys.

Focus & move

Alt+H/J/K/L
Focus window left / down / up / right
Alt+Shift+H/J/K/L
Swap window in that direction

Resize & layout

Alt+- / =
Shrink / grow focused window
Alt+Shift+=
Balance sizes
Alt+F
Fullscreen focused window
Alt+Shift+Space
Toggle float / tile (for apps that fight tiling)
Alt+/
Flip tile orientation (horizontal ↔ vertical)
Alt+,
Accordion layout (stack windows)
Alt+Shift+C
Reload Aerospace config

🧵tmux — inside your terminal

Prefix is Ctrl+A — press it, release, then press the next key.

Panes (splits within one tmux window)

Ctrl+Athen|
Split horizontally
Ctrl+Athen-
Split vertically
Ctrl+H/J/K/L
Focus pane left/down/up/right (vim-aware — also jumps into nvim splits)

Windows (tabs inside a session)

Ctrl+AthenC
New tmux window
Ctrl+Athen1 … 9
Jump to tmux window N
Ctrl+AthenTab
Last tmux window

Sessions & copy

Ctrl+AthenD
Detach session (keep running, return with tmux attach)
Ctrl+Athen[
Enter copy mode — scroll + select with vi keys (v select, y yank)
Ctrl+AthenR
Reload tmux config

⌨️Shell (zsh + atuin + fzf + zoxide)

Supercharged shell with fuzzy everything.

Search

Ctrl+R
Atuin — fuzzy search entire history, with timestamps and workdir context
Ctrl+T
Fuzzy file picker (inserts path into current command)
Alt+C
Fuzzy cd
↑ / ↓
Prefix-matched history search (type then press ↑)
→
Accept the greyed-out autosuggestion
Tab
Fuzzy completion picker

Jump around

z <fragment>
Jump to frecent directory (zoxide) — e.g. z flywheel
zi
Interactive directory picker

Useful aliases

ll / la
Long list / long list with dotfiles
.. / ...
cd .. up 1 / 2 levels
gs / gd / gco / gl
git status / diff / checkout / pretty log
reload
source ~/.zshrc

Built-in shortcuts most people don't know.

💻CLI — when something breaks or needs tweaking

aerospace reload-config
Reload Aerospace after editing ~/.config/aerospace/aerospace.toml
skhd --restart-service
Reload skhd after editing ~/.skhdrc
brew services restart felixkratz/formulae/sketchybar
Reload sketchybar after editing ~/.config/sketchybar/*
aerospace list-workspaces --focused
Which workspace am I on?
aerospace list-windows --all
What windows exist, on which workspaces, on which monitors?
aerospace list-monitors
Which monitors does Aerospace see?
Source of truth: edit files in ~/dotfiles/, not live configs. Copy to live location, then commit. On a new machine, run ~/dotfiles/mac/setup.sh to reproduce everything.

🔑Key legend

Cmd
⌘ Command (next to spacebar)
Opt
⌥ Option / Alt (next to Cmd — same key as "Alt" in all bindings)
Ctrl
⌃ Control
Shift
⇧ Shift
`
Backtick (top-left, under Esc)
[ ]
Square brackets (right of P)
\
Backslash (right of ])

Note: this cheatsheet calls "Opt" and "Alt" interchangeably — they're the same physical ⌥ key on your Mac keyboard.
